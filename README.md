# Japan: Hike & Ride

A trip planning page for a group hiking and cycling trip to Japan. Built to share with friends as a GitHub Pages site.

## To host on GitHub Pages

1. Create a new GitHub repository (e.g. `japan-trip`)
2. Upload `index.html` to the repository root
3. Go to **Settings → Pages → Source → Deploy from a branch → main / (root)**
4. Your page will be live at `https://yourusername.github.io/japan-trip`

## What's included

- All three trip options (Option A: Hike only, B1: + Shimanami, B2: + Biwaichi)
- Day-by-day itineraries for each
- Season guide with month-by-month breakdown
- Flight info for London and SF Bay Area
- Booking tips and logistics notes
